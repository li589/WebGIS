# 性能基线 — 风场 / 核显（Intel UHD）

> 环境参考：HP 暗影精灵 9，i9-13900HX，32GB，RTX4060；**Chrome 跑在 Intel UHD**。  
> 开启探针：URL 加 `?perf=1`，或 `localStorage.setItem('cgda.perf','1')` 后刷新。  
> 控制台抄录：`copy(JSON.stringify(window.__cgdaPerf(), null, 2))`

## 探针事件种类

- `dataVersion.bump` / `overlay.sync` / `overlay.syncMs`
- `tile.fetchMs` / `tile.viewportFillMs` / `tile.cacheSize`
- `wind.rafDt` / `wind.readPixels`（含 `skipped`）

## 本轮代码侧收口（F→B→E→D→A→C）

| Phase | 要点 |
|-------|------|
| F | `?perf=1` / `localStorage cgda.perf=1`，`window.__cgdaPerf()` |
| B | dataVersion 100ms coalesce；mergeCache 不再全清；child prefetch priority=2，pending 高时暂停 |
| E | 瓦片 LRU 96 + 丢弃 \|Δz\|>1；风+标量同屏时标量回退 MapLibre fill |
| D | Open-Meteo API 槽 4；sync `get_or_generate_tile_sync` 线程闸与 async 对齐 |
| A | Intel/`?windgl=lite` → 48²；idle 30fps + 隔帧 readPixels；减少 triggerRepaint |
| C | hour debounce 100ms；crossfade token/checksum 加固 |

长时间多图层：建议关闭不可见表层，避免缓存与 WebGL 目录堆积。
