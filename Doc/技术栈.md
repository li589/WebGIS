## 启动与基础设施

- 一键联调：仓库根目录 `launch.py` / `start.bat`（Windows）/ `start.sh`（Linux）
  - `python launch.py start` — 运行栈 + FastAPI + Celery + 前端
  - `python launch.py sync` — 数据面同步（默认 `open-meteo-sync`）
  - `python launch.py stop` / `status` / `flush`
- 运行栈：`Code/backend/docker-compose.yml`（Redis `:6379`，MinIO `:9100/:9101`，Open-Meteo API `:8080`）
- 数据栈：`Code/infra/data-sync/`（named volume，不进项目目录）
- 本地 Node/Python 运行时可参考 `Env/`
