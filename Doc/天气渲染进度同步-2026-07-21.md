## 1. 本轮结论

| 主题 | 结论 |
|------|------|
| 叠框（高纬更重） | 根因是「度数外扩 + 父子 z 全量叠绘」在 Web Mercator 下屏幕放大；已改为全球格网贴合 + 半开格心归属 |
| 空隙（横/纵缝） | 根因是「格元裁进瓦片半开框」而格网边 ≠ Mercator 瓦片边；已取消格元×瓦片求交 |
| 风粒子 | WebGL 主路径密度/拖尾已抬升；Canvas `?windgl=0` 仍可用 |
| 标量色场 | `FILL_OPACITY_MAX=0.90`；降水近零透明；湿度 greens 加深 |

## 2. 几何契约（前后端对齐）

### 2.1 全球格网

- 格心：`(i + 0.5) * resolution`（经度 / 纬度各自独立）
- 分辨率：`zoom_to_resolution(z)` / 前端 `weatherZoomToResolution(z)` 对齐
- 后端：`field_mapping.aligned_grid_axes` / `lon_centers_half_open` / `lat_centers_half_open`
- 前端：`weather-grid-lattice.ts`（`snapToLatticeCenter` / `latticeCellBounds`）

### 2.2 半开瓦片归属（仅约束格心，不裁格元多边形）

```
west ≤ lon < east
south < lat ≤ north
```

- 后端：`point_in_tile_half_open` + `_stamp_and_clip_tile_geojson`
- 前端：`pointInTileHalfOpen` + `filterGeojsonInsideTileBounds`
- **格元矩形**：由格心生成半格贴合全球矩形；邻格共享边；允许轻微越过瓦片地理框（由半开归属保证每格只发一次）
- **禁止**：把格元 `intersect` 进瓦片框（会在瓦片边≠格网边时挖出横/纵空隙）

### 2.3 合并与垫底

| 规则 | 行为 |
|------|------|
| 同级合并 | Point 按量化 lon/lat + height 去重（first-wins） |
| 父级 underlay | 仅当本级覆盖率 `< 1`；只用 `z-1`；裁掉已落在本级已加载瓦片 bbox 内的点 |
| 合并缓存 | key 含覆盖签名（命中瓦片列表），瓦片到达后不会复用半加载叠层结果 |
| 补洞 | `GAP_SWEEP` ~2.5s；zoom debounce 缩短 |

## 3. 渲染链路（前端）

```text
/weather/tiles → weather-tile-manager（调度/半开裁点/父级补洞）
              → mergeWeatherTiles
              → prepareContinuousFieldGeojson / geojsonPointsToGridCells（全球格元）
              → MapLibre fill | WebGL/Canvas 风粒子
```

关键文件：

| 文件 | 职责 |
|------|------|
| `stores/weather-tile-manager.ts` | 视口调度、半开裁点、父级 gap-fill、merge cache |
| `services/weather-tile-utils.ts` | 合并去重、半开过滤 |
| `components/map/weather-grid-lattice.ts` | 格网数学（与后端对齐） |
| `components/map/weather-overlay-renderers.ts` | 点→格元、grid_fill 同步 |
| `components/map/weather-render.ts` | 色带 / 降水 alpha 表达式 |
| `components/map/wind-particle-webgl-*.ts` | WebGL 粒子主路径 |
| `components/map/wind-particle-canvas.ts` | Canvas 回退 |

## 5. 回归清单（人工）

1. Ctrl+F5 后开气温 / 降水 / 湿度：邻瓦应无叠框外框，高纬亦然  
2. 多级缩放（尤其分辨率跳变附近）：横/纵不应再有稳定空缝；短暂空洞应在补洞后消失  
3. 风场 WebGL：粒子连续；`?windgl=0` Canvas 可用  
4. 若仍有空缝：区分「大方格、等一会还在」（缺瓦 / Network）vs「发丝缝、一缩放就没」（抗锯齿）
