# 代码事实同步文档（2026-07-15）

本文档记录 2026-07-15 相对于上一版（2026-07-06）的代码变更事实，覆盖前端模块。

## 1. 本轮变更总览

本轮变更包含 7 项功能修改，涉及前端 8 个文件：

## 2. 详细变更

### 2.1 风场粒子流显示优化

**文件**: `frontend/src/components/map/wind-particle-canvas.ts`

**问题**: 海域风曲线少、密集区域路径互相交错不稳定

**修改**:
- `computeParticleCountForGrid`: 粒子密度系数 `area * 7` → `area * 5`，减少密集区域交错
- `resolveParticleCountForZoom`: 低缩放级别上限下调 200/400/800 → 160/320/600
- 新增 `MIN_WIND_SPEED_FOR_RENDER = 0.5`（m/s）静风阈值：低于此值的粒子重置到随机位置，避免海洋静风带粒子停滞堆积
- `DEFAULT_PARTICLE_OPTIONS`: particleCount 1000→800, maxAge 90→110, speedScale 0.035→0.028, fadeAlpha 0.018→0.024, lineWidth 1.4→1.2
- `TRAIL_LENGTH`: 28 → 32（更长拖尾，曲线更平滑）
- 年龄阶段参数: AGE_BAND_YOUNG_RATIO 0.15→0.12, AGE_BAND_OLD_RATIO 0.75→0.78, AGE_BAND_ALPHAS [0.28,1.0,0.38]→[0.22,1.0,0.30]

### 2.2 工作流状态面板修复

**文件**: `frontend/src/stores/layers/types.ts`, `frontend/src/stores/layers/index.ts`, `frontend/src/components/workflow/WorkflowStatusPanel.vue`

**问题**: 面板列表不显示失败/排队的工作流（活跃图层被移除后工作流记录仍在但不显示）

**修改**:
- `JobLayerItem` 接口新增 `catalogId?: string` 字段
- `upsertJobLayer` 方法自动将 catalogId 注入到 JobLayerItem
- `WorkflowStatusPanel.vue` 的 `workflowItems` computed 合并 `jobLayers` 中的孤儿工作流（不在 activeLayersDisplay 中的工作流），通过 `layerLibrary` 查找元信息

### 2.3 图层符号系统配色方案切换

**文件**: `frontend/src/components/map/weather-render.ts`, `frontend/src/components/InfoPanel.vue`, `frontend/src/components/map/wind-particle-overlay-controller.ts`, `frontend/src/stores/layers/types.ts`, `frontend/src/stores/layers/index.ts`

**功能**: 类似 ArcGIS 的符号系统配色方案切换，适配点/线/面/风场/粒子流/热力图/栅格等

**修改**:
- `WEATHER_PALETTES` 从 4 种扩展到 14 种配色方案（热力橙红/降水青蓝/风场蓝/品红黄/Viridis 科学/光谱彩虹/渐变蓝/渐变红/渐变绿/黄红外推/蓝绿渐变/红蓝发散/紫橙渐变/暗色彩虹）
- 每种配色方案新增 `label`（显示名）和 `type`（sequential/diverging/qualitative）元数据
- 导出 `WEATHER_PALETTE_OPTIONS` 数组供 UI 使用
- 新增 `paletteToParticleColors()` 函数：将配色方案线性插值为 12 色粒子色阶
- `WeatherPaletteDefinition` 接口新增 `label` 和 `type` 字段
- `ActiveLayer` 和 `ActiveLayerDisplay` 新增 `paletteOverride?: string | null` 字段
- `layersStore` 新增 `setLayerPaletteOverride(instanceId, palette)` 方法
- `activeLayersDisplay` computed 中将 paletteOverride 应用到 renderHint.palette
- `WindParticleCanvas` 新增 `setColors(colors: string[])` 方法，支持动态更新粒子色阶
- `WindParticleOverlayController` 在 sync 时通过 `paletteToParticleColors` 应用当前配色到粒子画布
- `InfoPanel.vue` 新增配色方案选择器 UI（下拉列表，色带预览，类型标签，恢复默认按钮）

### 2.4 标题栏工作流状态按钮去冗余

**文件**: `frontend/src/components/workflow/WorkflowStatusButton.vue`

**修改**: label computed 去掉数字，只保留状态文字（"运行中"/"失败"/"完成"），数字由 badge 显示

### 2.5 图层分类重组

**文件**: `frontend/src/stores/layers/catalog.ts`, `frontend/src/stores/layers/index.ts`

**修改**:
- `LAYER_CATEGORIES` 重组为 8 个新分类：online-weather(在线天气,15个), climate(气候与历史,3个), disaster(灾害监测,2个), atmosphere(大气环境,1个), vegetation(植被与土地,5个), terrain(地形与遥感,4个), research-group(课题组数据,6个), boundary(行政边界,1个)
- `LAYER_LIBRARY` 中 37 个图层的 category 字段全部更新
- `index.ts` 中默认 fallback category `'runtime'` → `'research-group'`

### 2.6 WorkflowStatusPanel 全面增强

**文件**: `frontend/src/components/workflow/WorkflowStatusPanel.vue`

**修改**: 完整重写，新增标题栏统计（总计/活跃/成功率）、整体进度条、天气瓦片分图层状态+全局缓存命中率、分类统计、工作流列表项增强（commandType/metrics/创建时间/运行时长/结果链接/可展开诊断）

### 2.7 定位按钮位置调整

**修改**:
