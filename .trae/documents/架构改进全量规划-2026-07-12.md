定义 TileProvider Protocol 和 TileResponse 数据类：

```python
from __future__ import annotations
from dataclasses import dataclass
from typing import Protocol, Any

@dataclass
class TileResponse:
    """统一瓦片响应容器。"""
    data: bytes
    content_type: str
    cache_status: str | None = None
    extra_headers: dict[str, str] | None = None

class TileProvider(Protocol):
    """瓦片提供者协议。"""
    def matches(self, layer_id: str) -> bool: ...
    async def get_tile(
        self,
        layer_id: str,
        z: int,
        x: int,
        y: int,
        **params: Any,
    ) -> TileResponse: ...
```

**文件 2：`Code/backend/app/services/tile_provider_registry.py`**

Provider 注册表，按注册顺序匹配：

```python
from __future__ import annotations
from app.services.tile_provider_protocol import TileProvider, TileResponse

class TileProviderRegistry:
    def __init__(self) -> None:
        self._providers: list[TileProvider] = []

    def register(self, provider: TileProvider) -> None:
        self._providers.append(provider)

    def resolve(self, layer_id: str) -> TileProvider | None:
        for provider in self._providers:
            if provider.matches(layer_id):
                return provider
        return None

    async def get_tile(self, layer_id: str, z: int, x: int, y: int, **params) -> TileResponse:
        provider = self.resolve(layer_id)
        if provider is None:
            raise ValueError(f"No tile provider matches layer_id: {layer_id}")
        return await provider.get_tile(layer_id, z, x, y, **params)

tile_provider_registry = TileProviderRegistry()
```

```python
from app.services.tile_proxy_service import TILE_URL_TEMPLATES
from app.services.tile_provider_protocol import TileResponse

class BaseMapTileProvider:
    def matches(self, layer_id: str) -> bool:
        return layer_id in TILE_URL_TEMPLATES

    async def get_tile(self, layer_id: str, z: int, x: int, y: int, **params) -> TileResponse:
        from app.services.tile_proxy_service import tile_proxy_service
        data = await tile_proxy_service.fetch_tile(
            tile_id=layer_id, x=x, y=y, z=z, use_cache=params.get("use_cache", True)
        )
        # 推断 content_type
        if "satellite" in layer_id or "img" in layer_id:
            content_type = "image/jpeg"
        else:
            content_type = "image/png"
        return TileResponse(
            data=data,
            content_type=content_type,
            extra_headers={"Cache-Control": "public, max-age=86400", "X-Tile-Provider": layer_id},
        )
```

```python
from __future__ import annotations
import json
from app.services.tile_provider_protocol import TileResponse

class WeatherTileProvider:
    def __init__(self) -> None:
        self._weather_layer_ids: set[str] | None = None

    def _ensure_layer_ids(self) -> set[str]:
        if self._weather_layer_ids is None:
            from app.services.layer_catalog import get_layer_catalog
            catalog = get_layer_catalog()
            self._weather_layer_ids = {
                layer.layer_id for layer in catalog.list_layers()
                if layer.source_type == "weather"
            }
        return self._weather_layer_ids

    def matches(self, layer_id: str) -> bool:
        return layer_id in self._ensure_layer_ids()

    async def get_tile(self, layer_id: str, z: int, x: int, y: int, **params) -> TileResponse:
        from app.weatherengine.tile_service import get_weather_tile_service
        from app.core.config import settings
        geojson, cache_status = await get_weather_tile_service().get_tile(
            layer_id=layer_id, z=z, x=x, y=y,
            hour=params.get("hour", 0), model=params.get("model"),
        )
        return TileResponse(
            data=json.dumps(geojson, ensure_ascii=False).encode(),
            content_type="application/geo+json",
            cache_status=cache_status,
            extra_headers={
                "Cache-Control": f"public, max-age={settings.weather_cache_ttl_seconds}",
                "X-Weather-Tile-Cache": cache_status,
            },
        )
```

统一瓦片路由：

```python
from fastapi import APIRouter, HTTPException, Query, Response
from app.services.tile_provider_registry import tile_provider_registry

router = APIRouter(prefix="/unified-tiles", tags=["tiles"])

@router.get("/{layer_id}/{z}/{x}/{y}")
async def get_unified_tile(
    layer_id: str,
    z: int,
    x: int,
    y: int,
    hour: int | None = Query(default=0, ge=0, le=47),
    model: str | None = Query(default=None),
    use_cache: bool = True,
) -> Response:
    try:
        tile = await tile_provider_registry.get_tile(
            layer_id, z, x, y, hour=hour, model=model, use_cache=use_cache
        )
    except ValueError as exc:
        raise HTTPException(status_code=404, detail=str(exc)) from exc
    except Exception as exc:
        raise HTTPException(status_code=503, detail=f"Tile unavailable: {exc}") from exc

    headers = dict(tile.extra_headers or {})
    if tile.cache_status:
        headers["X-Tile-Cache"] = tile.cache_status
    return Response(content=tile.data, media_type=tile.content_type, headers=headers)
```

```python
"""导出 FastAPI OpenAPI schema 到 JSON 文件。"""
import json
import sys
from pathlib import Path

def export_openapi(output_path: str | None = None) -> None:
    sys.path.insert(0, str(Path(__file__).resolve().parent.parent))
    from app.main import app
    schema = app.openapi()
    if output_path is None:
        output_path = str(Path(__file__).resolve().parent.parent.parent / "frontend" / "openapi.json")
    Path(output_path).write_text(json.dumps(schema, indent=2, ensure_ascii=False), encoding="utf-8")
    print(f"OpenAPI schema exported to {output_path}")

if __name__ == "__main__":
    export_openapi(sys.argv[1] if len(sys.argv) > 1 else None)
```

## 六、假设与决策

### 6.1 假设

1. FastAPI 默认在 `/openapi.json` 暴露完整 OpenAPI schema（已在 `main.py` 中确认未禁用）
2. `tile_proxy_service` 的 `TILE_URL_TEMPLATES` 字典 key 集合即为全部合法底图 provider ID
3. `layer_catalog` 的 `list_layers()` 可返回所有已注册图层，`source_type == "weather"` 可区分天气图层
4. 前端 `runtime-api.ts` 中的 API 调用函数（非类型定义）不依赖具体类型结构，仅作为参数/返回值标注

### 6.2 决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| Phase 4 `__init__.py` 遮蔽修复 | 仅修测试，不改 `__init__.py` | `main.py` 依赖 re-export；改别名增加无谓变更 |
| Phase 5 旧端点处理 | 标记 `deprecated=True`，不删除 | 向后兼容，前端迁移完成后再清理 |
| Phase 5 Provider 匹配策略 | 按注册顺序线性匹配 | 底图 provider ID 与天气 layer_id 无交集，无需复杂匹配 |
| Phase 5 天气 provider layer_id 缓存 | 首次调用时惰性加载并缓存 | 避免启动时阻塞；layer_catalog 不变 |
| Phase 6 UI 专属类型 | 保持手写，不移入自动生成 | 自动生成仅覆盖后端 Pydantic 模型；UI 类型无后端对应 |
| Phase 6 re-export 策略 | 通过 `api-reexports.ts` 扁平化 | 保持现有 `import { X } from 'runtime-api'` 路径兼容 |
| Phase 6 OpenAPI JSON 存放位置 | `Code/frontend/openapi.json` | 前端可直接访问，git 可追踪 |

## 八、风险与缓解

| 风险 | 概率 | 缓解 |
|------|------|------|
| Phase 5 天气 provider layer_id 缓存过期 | 低 | layer_catalog 在运行时不变；如需刷新可加 `invalidate()` 方法 |
| Phase 5 前端瓦片 URL 替换遗漏 | 中 | 用 Grep 全局搜索 `/tiles/` 和 `/weather/tiles/` 确认无遗漏 |
| Phase 6 自动生成类型与手写类型字段名不一致 | 中 | `openapi-typescript` 默认使用 snake_case（与后端一致）；前端现有类型也用 snake_case，应兼容 |
| Phase 6 `openapi-typescript` 版本兼容性 | 低 | 使用 v7.x 稳定版 |
| Phase 6 生成文件过大 | 低 | FastAPI 项目模型数量可控（~30 个），生成文件约 500-1000 行 |
